这道题需要读位运算比较了解。

当找不到思路，可以尝试先把问题简化一下，题目说有两个不一样的数，假如只有一个呢？

如果只有一个，怎么可以找到？

如果只有一个是不同的，其他都是相同的，如果能把相同的都抵消掉，不就剩下唯一一个不同的了。
怎么可以抵消相同的数呢？如果对位运算熟悉，就能想到可以利用异或运算，如果两个数相同，
异或的结果就为 0.

那么，初始化 a = 0，然后和所有的数字进行异或，相同的数异或还是 0，最后就剩下唯一一个不同的数。

而题目中有两个不同的数，如果对所有数字进行异或，那就会剩下两个数。如果可以把这两个数字拆开，
分到两个不同的数组中，就可以转化成只有一个数不同的情况。

那么怎么进行拆分呢？需要找到一个拆分的依据。

假设两个不同的数分别位 x ，y，初始化 a = 0, 遍历所有元素进行异或，相同的元素都因为异或而抵消，
最后就会得到 a = x \asciicirc y . 因为 x 和 y 不同，那么 x \asciicirc y 不可能等于 0，那么就
意味着 x \asciicirc y 对应的二进制表示中，一定有至少 1 位是 1，设这个位置是第 n 位，
对于 x 和 y 来说, 因为 x \asciicirc y 不等于 0，第 n 位肯定是一个为 1， 一个为 0，不可能都是 1.
因此，可以找到 x \asciicirc y 中为 1 的第 n 位，然后根据它拆分 x 和 y，对于数组中相同的数，也是根据
第 n 位是否为 1 拆分，相同的数字第 n 位肯定相同，也就会被拆分到相同的数组。最终就可以得到两个数组，
x ，y 分别在两个不同的数组中，此时就转化成数组中只有一个不同的元素，就能通过异或得到 x 和 y 了。
