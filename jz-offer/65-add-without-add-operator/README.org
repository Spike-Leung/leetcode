不能用四则运算，那么还有什么可以进行数字操作的呢？ 容易想到，可以尝试位运算。

要怎么利用位运算呢？位运算常见的操作有与，或，异或，左移，右移，哪些操作能进行计算呢？

将数字转换成二进制，将它们相加，寻找规律。

可以发现，对于第 i 位，相加后的值对应异或运算的结果。

对于进位，相当于与运算的结果，而且因为进位了，还需要左移一位。

另外，数字在 JS 中是以 补码 形式存储的的，加减法可以统一处理，就不用担心减法的问题了。
