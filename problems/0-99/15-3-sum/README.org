[[https://leetcode-cn.com/problems/3sum/solution/3sumpai-xu-shuang-zhi-zhen-yi-dong-by-jyd/][三数之和（排序+双指针，易懂图解）]]

找三个数之和，如果固定一个数，那么就是找两个数之和，如果再固定一个数，
就是在剩下的数字里找最后一个数字。

既然涉及从数组中查找，最好还是先将数组排序。

排序后，遍历数组，依次去固定一个数，再从剩下的数中找另外两个数。由于已经
排序，可以利用双指针，从剩余数字的头尾往中间遍历，通过控制头尾指针，可以
调整和的大小。当固定的数和头尾指针的数的和为 0，则记录当前的三个数字。当
他们的和大于 0，则缩小右指针，让和变小。重复这样的判断，直到找到所有组合。

由于题目中要求是不重复的组合，当固定数和左右指针碰到重复数字时，需要跳过处理。
