[[https://leetcode-cn.com/problems/chou-shu-lcof/solution/mian-shi-ti-49-chou-shu-dong-tai-gui-hua-qing-xi-t/][剑指 Offer 49. 丑数（动态规划，清晰图解）]]底下有个高赞的评论不错。


观察可以发现，丑数可以由前面的数字，乘以2，3，5得到。

又因为需要按照从小到达的顺序排，因此乘以2，3，5 后得到的数字，应该选最小的先放在后面。

如果数字已经乘以 2 或 3 或 5，就不需要再乘一次了，因此需要记录哪个数字已经处理过，就需要三个指针去对应。

当计算得到一个丑数后，如果是乘以 2 得来的，那 2 的指针后移一位，指向下一个没有处理数字。
如果可能是乘以 2 得到的，也肯能是乘以 3 得到的，则 2 和 3 的指针同时后移一位，指向下一个没有处理的数字。

最后返回第 n 个丑数即可。
