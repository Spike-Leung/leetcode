做这道题，可以逐步去考虑：
1. 如果没有 ~.~ 和 ~*~ 怎么匹配

   如果没有 ~.~ 和 ~*~ ，只要逐个字符比对字符串和表达式，全部匹配则符合，有一个不匹配则不符合。

2. 加上 ~.~ 之后怎么匹配
   如果有 ~.~, 那么它可以匹配字符串中的任意字符。

3. 加上 ~*~ 之后怎么匹配
   * 表示可以匹配 0 或任意次，需要分情况：

     1. 表达式中的字符和字符串不匹配，* 就没必要匹配了，也就是匹配 0 次
     2. 表达式中的字符和字符串匹配：
        1. 匹配 0 次
        2. 当作一个字符，匹配 1 次
        3. 匹配任意次

        由于可能匹配任意次，字符串可能匹配完了，判断时需要考虑越界的问题。


根据以上的分析，对应几个状态，编写代码即可。

另外，这道题可以用状态机表示，可以用一个具体的例子，包含 . 和 * 去画一个状态机
的图，便于分析问题。

自己做的时候，对于 * 的考虑欠缺了，只考虑了可以任意次匹配的情况，忽略了 0 次 和 1 次的情况，
用状态机去描述的话，可以减少疏忽。
