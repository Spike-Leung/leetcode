* 思路
如果往一个字符串数组，插入某个（一个）字符串数组（可以为空），能够得到另一个字符串数组，则认为是相似的。

考虑满足条件的情况：

- 都往左边插入
- 都往右边插入
- 插入在中间

会发现，这三种情况：

- 都往左边插入： 后缀刚好就是长度较短的字符串数组
- 都往右边插入： 前缀刚好都是长度较短的字符串数组
- 插入在中间：前缀 + 后缀刚好就是长度较短的字符串

于是，可以统计相同的前缀和后缀的数量，从前往后遍历找相同的前缀个数，从后往前遍历找相同的后缀个数。

最后，判断前缀 + 后缀的个数，是否刚好等于较短字符串数组的长度。
