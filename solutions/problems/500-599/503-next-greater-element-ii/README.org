* 思路
  对于这种求下一个较大元素，一般考虑用单调栈。

  对于循环的问题，可以“复制”一份数组在后面，模拟循环，然后在这个两倍长
  度的数组上，构建单调递增栈，利用取模操作，去设置结果。
